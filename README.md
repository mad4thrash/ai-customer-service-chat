# AI Customer Service Chat

Monorepo containing both client and server for the AI-powered customer service chat application.

## Structure

- `/client` - React + TypeScript frontend
- `/server` - Node.js + TypeScript backend with AI agent

## Getting Started

### Server
```bash
cd server
npm install
npm run dev
```

### Client
```bash
cd client
npm install
npm run dev
```

## Features

- AI-powered customer service agent
- Real-time chat interface
- E-commerce product integration
